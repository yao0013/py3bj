营业收入 = 主营业务收入 + 其他业务收入
营业总收入 = 营业收入 + 利息收入 + 已赚保费 + 手续费及佣金收入


毛利润 = 营业收入 - 营业成本（即主营业务成本 + 其他业务成本）
毛利率 = 毛利润 / 营业收入

营业利润 = 毛利润 - 营业税金及附加 - 
           销售费用 - 管理费用 - 财务费用 - 
		   资产减值损失 + 公允价值变动收益(-公允价值变动损失) + 投资收益(-投资损失)
营业利润率 = 营业利润 / 营业收入

净利润 = 营业利润 + 营业外收入 - 营业外支出 - 所得税
净利润率 = 净利润 / 主营业务收入



国内不常用（另一套）：
	净营业收入 = 营业收入 - 营业税金及附加
	净收入（NI）Net Income = 营业利润 + 其他业务利润 - 管理费用 - 财务费用 + 投资收益 + 补贴收入 + 营业外收入 - 营业外支出
	利润（Profit）= 营业利润 + 营业外收入 - 营业外支出
	净利润 （Net Profit） = 利润 - 所得税
